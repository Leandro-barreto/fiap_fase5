# Monitoring

Esta pasta destina‑se a armazenar scripts e configurações relacionados
ao monitoramento da API e dos modelos de Machine Learning.  Por
exemplo, você pode colocar aqui arquivos de configuração do
Prometheus, dashboards do Grafana ou documentação sobre como coletar
e expor métricas.  Neste exemplo, o conteúdo foi deixado como um
placeholder para que você personalize conforme as necessidades do
seu projeto.
